# CORET – Product Specification (V1)

## Product Type

iOS-first native application built in SwiftUI.

CORET is a wardrobe operating system that measures structural cohesion and guides optimization.

---

# V1 Scope

## Core Areas

1. Wardrobe
2. Optimize
3. Profile

---

## 1️⃣ Wardrobe

### Features

- Image upload per item (manual, no AI detection)
- Required fields:
  - Category (Top / Bottom / Shoes / Outerwear)
  - Silhouette (Structured / Balanced / Relaxed)
  - Color (user-selected, mapped automatically)
  - Archetype tag (single selection)

### Display

- Satisfying grid layout
- Calm UI
- Structural status visible at top
- No aggressive analytics on main screen

---

## 2️⃣ Cohesion Score

Hybrid system:

Primary:
Structural Status (Structuring / Refined / Coherent / Aligned / Architected)

Secondary:
0–100 score

Breakdown available on tap.

---

## 3️⃣ Optimize View

### Primary Focus:
Future potential, not current weaknesses.

### Displays:

- Visual structural simulation
- Potential increase (e.g. +9)
- Priority 1 target
- Priority 2 target (Pro)

### Strategy:

- Add to strengthen (primary)
- Reconsider (secondary)

No shopping integration in V1.

---

## 4️⃣ Archetype System

- Primary archetype
- Secondary archetype
- Single archetype tag per item
- Archetype change allowed via profile
- New profile creation supported
- Wardrobe persists across profile changes

---

## 5️⃣ Seasonal Recalibration

- Location-based season detection
- Suggest recalibration (not forced)
- Generates new seasonal roadmap
- Adjusts structural weight temporarily

---

## 6️⃣ Structural Evolution

Narrative-based evolution tracking.
Not traditional score graph.

Phases displayed over time.

---

## 7️⃣ Monetization

Freemium model.

Free:
- Basic score
- 1 active target
- Limited optimize preview

Pro:
- Full simulation
- Multiple roadmap targets
- Advanced analytics
- Drift tracking
- Structural evolution detail

Target pricing: 9–12 USD/month

---

## Explicitly NOT in V1

- Machine learning
- Auto color detection
- Retail integrations
- Social features
- Budget tools
- Cross-platform support
