Light mode
Design tokens
Uses the exact same token system as Wardrobe, Studio, and Discover. Dark: warm blacks + gold (#C9A96E). Light: cream + darker gold (#b8860b). Glass cards, gradient dividers, ambient blobs.
Navigation
Profile is not a tab — accessed via menu icon on any screen. "Tilbake" returns to previous tab. Tab bar shows 3 tabs, none highlighted. Same tab bar component across all screens.
Content hierarchy
1. Style Context hero (identity at a glance)
2. Structural Identity (archetype editor)
3. Season (with recalibration)
4. Favorite Fits (outfit showcase)
5. Import Sources (data connections)
6. Settings (system config)
Backend API
GET/PUT /api/profile — style_context + archetype. Season recalibration triggers full engine recompute. Archetype changes recompute Cohesion + Optimize.
